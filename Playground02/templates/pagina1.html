Página Complexa

Aqui a gente pode ver a combinação de vários tags, criando uma página com vários elementos.

Para isso, eu coloquei o HTML num arquivo separado, para não ocupar muito espaço no código Python.

Exemplos de Tags

O HTML tem suporte a vários tipos de elementos. Vamos ver alguns dos mais básicos.

  • h1, h2, h3, etc: coloque títulos e subtítulos entre esse tags
  • p: coloque o texto do parágrafo entre tags p
  • a: use esse tag com a propriedade href="http://endereco.com" para criar links
  • ul + li: abra uma lista ul e coloque items li dentro dela
  • img: use esse tag com a propriedade src="http://endereco.com/foto.jpg" para criar imagens

Outras Tecnologias da Web

O visual padrão desses elementos todos pode ser modificado usando uma outra linguagem: CSS. Se quiser saber mais sobre ela, clique neste link.

Já para recursos mais avançados, como atualizações na página ou a interatividade do usuário (ex: clique em botões), usamos o Javascript. Para saber mais sobre o assunto, acesse esta página.
